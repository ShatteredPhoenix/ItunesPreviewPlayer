ItunesPreviewPlayer

An application made in Swift intergrating the Itunes Search API with JSON to allow users search to search the iTunes store for albums and preview the tracks inside them.

Known Issues:

-- Some simple constraint issues when album name is too long

-- Button issue where stop button will not change back to 'play' when priview of track finishes

-- Another button issue, where icon won't change back to 'play' when another track is selected on top of a track already being played, the previous button will not change back to the 'play' icon.
